Cookie Policy

DELTA Plant defaults to necessary cookies only. Optional categories stay off until you explicitly enable them.

NASA Monitor

Cookie Categories

Category Purpose Default
Necessary JWT session, CSRF token, security boundaries, rate-limiting context. Always on
Maps Enables external tile requests for interactive field visualization. Off
Voice Allows voice synthesis requests for field reports. Off
LLM Allows large language model narratives in addition to the procedural diagnosis. Off
Analytics Reserved for future privacy-safe product telemetry if enabled by the user. Off

How Consent Works

  • The browser stores a public anonymous token so you can manage preferences without creating an account.
  • The backend stores only a hashed form of that token inside an encrypted consent registry.
  • Changing consent updates both the UI cookie and the server-side encrypted record.
